31.10.3 match_­results size [re.results.size]

size_type size() const;

1
#

Returns: One plus the number of marked sub-expressions in the regular expression that was matched if *this represents the result of a successful match. Otherwise returns 0. [ Note: The state of a match_­results object can be modified only by passing that object to regex_­match or regex_­search.
