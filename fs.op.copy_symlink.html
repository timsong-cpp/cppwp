27.10.15.5 Copy symlink [fs.op.copy_symlink]

void copy_symlink(const path& existing_symlink, const path& new_symlink); void copy_symlink(const path& existing_symlink, const path& new_symlink, error_code& ec) noexcept;

1

Effects: Equivalent to function(read_symlink(existing_symlink), new_symlink) or
function(read_symlink(existing_symlink, ec), new_symlink, ec), respectively, where function is create_symlink or create_directory_symlink, as appropriate.

2

Throws: As specified in [fs.err.report].
