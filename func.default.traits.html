23.14.16 Default functor traits [func.default.traits]

namespace std {
  template <class T = void>
  struct default_order {
    using type = less<T>;
  };
}
1
#

The class template default_­order provides a trait that users can specialize for user-defined types to provide a strict weak ordering for that type, which the library can use where a default strict weak order is needed. For example, the associative containers and priority_­queue use this trait.

#

namespace sales {

    int id;

    }
  };

namespace std {
