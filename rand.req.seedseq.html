1
#
A seed sequence is an object that consumes a sequence of integer-valued data and produces a requested number of unsigned integer values i, 0â‰¤i<232, based on the consumed data.

2
#
A class S meets the requirements of a seed sequence if the expressions shown in Table 98 are valid and have the indicated semantics, and if S also meets all other requirements of [rand.req.seedseq].
In Table 98 and throughout this subclause:
  • (2.1)
    T is the type named by S's associated result_type;
  • (2.2)
    q is a value of type S and r is a value of type S or const S;
  • (2.3)
    ib and ie are input iterators with an unsigned integer value_type of at least 32 bits;
  • (2.4)
    rb and re are mutable random access iterators with an unsigned integer value_type of at least 32 bits;
  • (2.5)
    ob is an output iterator; and
  • (2.6)
    il is a value of type initializer_list<T>.
