5
#
These functions compute a non-conjugated dot product with an explicitly specified result type.
6
#
Returns: Let N be v1.extent(0).
  • (6.1)
    init if N is zero;
  • (6.2)
    otherwise, GENERALIZED_SUM(plus<>(), init, v1[0]*v2[0], …, v1[N-1]*v2[N-1]).
