iterator​::​iterator_­concept is defined as follows:
  • (1.1)
    If Base models random_­access_­range, then iterator_­concept denotes random_­access_­iterator_­tag.
  • (1.2)
    Otherwise, if Base models bidirectional_­range, then iterator_­concept denotes bidirectional_­iterator_­tag.
  • (1.3)
    Otherwise, if Base models forward_­range, then iterator_­concept denotes forward_­iterator_­tag.
  • (1.4)
    Otherwise, iterator_­concept denotes input_­iterator_­tag.

    Otherwise, iterator_­category denotes C.
