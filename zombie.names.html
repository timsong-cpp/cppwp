  • (2.1)
    argument_­type,
  • (2.2)
    first_­argument_­type,
  • (2.3)
    io_­state,
  • (2.4)

  • (2.6)
    seek_­dir.
3
#
The name stossc is reserved as a member function for previous standardization, and may not be used as a name for function-like macros in portable code.

#
