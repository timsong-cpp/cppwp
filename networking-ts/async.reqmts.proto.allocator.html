13.2.1 Proto-allocator requirements [async.reqmts.proto.allocator]

1

A type A meets the proto-allocator requirements if A is CopyConstructible (copyconstructible), Destructible (destructible), and allocator_traits<A>::rebind_alloc<U> meets the allocator requirements (allocator.requirements), where U is an object type. [ Note: For example, allocator<void> meets the proto-allocator requirements but not the allocator requirements.  â€” end note ] No comparison operator, copy operation, move operation, or swap operation on these types shall exit via an exception.
