16.5.3.4 Cpp17Hash requirements [hash.requirements]

1
#
A type H meets the Cpp17Hash requirements if:
  • (1.1)
    it is a function object type ([function.objects]),
  • (1.2)
    it meets the Cpp17CopyConstructible (Table 27) and Cpp17Destructible (Table 30) requirements, and
  • (1.3)
    the expressions shown in Table 32 are valid and have the indicated semantics.
2
#
Given Key is an argument type for function objects of type H, in Table 32 h is a value of type (possibly const) H, u is an lvalue of type Key, and k is a value of a type convertible to (possibly const) Key.
