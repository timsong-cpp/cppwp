31.12.8.5 Enum class perm_options [fs.enum.perm.opts]

1
#
The enum class type perm_options is a bitmask type ([bitmask.types]) that specifies bitmask constants used to control the semantics of permissions operations, with the meanings listed in Table 150.
The bitmask constants are bitmask elements.
In Table 150 perm denotes a value of type perms passed to permissions.
