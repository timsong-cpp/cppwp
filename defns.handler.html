16 Library introduction [library]

16.3 Definitions [definitions]

16.3.12[defns.handler]
handler function

non-reserved function whose definition may be provided by a C++ program
[ Note
:
A C++ program may designate a handler function at various points in its execution by supplying a pointer to the function when calling any of the library functions that install handler functions ([language.support]).
— end note
 ]
