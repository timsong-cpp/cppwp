23.3.3 Alias template make_­integer_­sequence [intseq.make]

template<class T, T N> using make_integer_sequence = integer_sequence<T, see below>;

1
#

If N is negative the program is ill-formed. The alias template make_­integer_­sequence denotes a specialization of integer_­sequence with N template non-type arguments. The type make_­integer_­sequence<T, N> denotes the type integer_­sequence<T, 0, 1, ..., N-1>. [ Note: make_­integer_­sequence<int, 0> denotes the type integer_­sequence<int>  â€” end note ]
